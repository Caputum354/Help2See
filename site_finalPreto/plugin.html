Tecnologia

O que está por dentro do plugin

Cada recurso é alimentado por um modelo de IA especializado — todos rodando em servidores brasileiros, em conformidade com a LGPD.

Voz Neural com SSML

Motor de síntese de voz baseado em modelos Tacotron 2 com prosódia adaptativa. Detecta pontuação, ênfase e contexto semântico para entonação natural em PT-BR. Velocidade, tom e volume ajustáveis pelo usuário em tempo real.

IA Generativa

Visão Computacional para Alt Text

Pipeline de visão baseado em modelo multimodal fine-tuned para contexto web brasileiro. Descreve imagens, gráficos, captchas, infográficos e banners com precisão semântica — e não apenas rótulos genéricos de objeto.

Visão Computacional

Simplificação com NLP

Modelo de linguagem especializado em reescrita simplificada para PT-BR. Reduz jargões jurídicos, técnicos e burocráticos para linguagem de leitura fácil (ABNT NBR 15290), mantendo a precisão do conteúdo original.

NLP

Monitoramento de Regressão

Crawler automatizado que revisa o DOM do site a cada atualização de conteúdo, detecta novas barreiras de acessibilidade e envia alertas em tempo real para o painel de conformidade — sem nenhum esforço do desenvolvedor.

Automação

Segurança e Privacidade

Zero armazenamento de dados pessoais dos usuários. Toda a inferência de IA ocorre em servidores brasileiros. Conformidade com LGPD garantida — nenhum conteúdo do site é retido além da sessão ativa.

LGPD Compliant
Instalação — 1 linha de código 
<!-- Cole antes de </body> -->

<script
  src="https://cdn.help2see.com.br/v2/plugin.min.js"
  data-key="SUA_CHAVE_AQUI"
  data-lang="pt-BR"
  defer
></script>

<!-- Opcional: configuração avançada -->

<script>
  window.H2SConfig = {
    position: 'bottom-right',
    theme:    'dark',
    voice:    'neural-v2',
    contrast: true
  };
</script>
